The ultimate boilerplate for hybrid apps

Luego de tropezar y tropezar, primero con jQuery mobile (shh! no le digan a nadie que alguna vez usamos jQuery mobile) y luego con jQmobi + BackBone. Decidimos hacer nuestro propio boilerplate para las apps hibridas. Este recoge toda nuestra experiencia y la pone a dispcición de todos. Enjoy!

Features

Phonegap

Lo bueno de usar phonegap es que podemos escribir el app una sola vez y publicarla facilmente en los diferentes tiendas de aplicaciones

Angular

Con Angular todo se vuelve super-modular. Las directivas, factories y services llegan al rescate para hacer aplicaciones escalables y mantenibles. Fastapps no le tiene miedo a los ng-*. Para más información visita Angularjs website

Less

Ya es imposible escribir CSS sin un preprocesador. Fastapps usa LESS para facilitar el cambio de colores de toda el app. Si necesitas mas información sobre less vente pa' ca'.

jQuery

Tienes algun plugin de jQuery que no puedes dejar de usar? Fastapps es compatible con jQuery. Solo ten cuidado de no usarlo erroneamente. De hecho el bundle ya tiene varios plugins integrados.

Gruntjs

Fastapps ya tiene integrado Gruntjs para que nos sea muy facil compilar los archivos LESS a CSS. Tambien podemos concatenar y minificar nuestos archivos javascript; esto nos permite tener una estructura de carpetas libres. PUM! algo cambió? Grunt compila todo. Sobre grunt

Download

Simplemente clona este repositorio escribiendo git clone https://github.com/juanpasolano/angular-mobile-demo.git en tu consola.

El clone del repo descarga dos branches importantes:
Master: Tiene el boileplate para comenzar a desarrollar
Development: Tiene el demo mostrando todas las directivas

Una vez clonado da npm install para instalar las dependencias de gruntjs.

Luego escribe grunt para empezar a vigilar y compilar los archivos less y contatenar y minificar los archivos javascript.
