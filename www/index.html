The ultimate boilerplate for hybrid apps

Luego de tropezar y tropezar, primero con jQuery mobile (shh! no le digan a nadie que alguna vez usamos jQuery mobile) y luego con jQmobi. Decidimos hacer nuestro propio boilerplate para las apps hibridas. Este recoge toda nuestra experiencia y la pone a dispcición de todos. Enjoy!

Features

Phonegap

Lo bueno de usar phonegap es que podemos escribir el app una sola vez y publicarla facilmente en los diferentes tiendas de aplicaciones

Angular

Con Angular todo se vuelve super-modular. Las directivas, factories y services llegan al rescate para hacer aplicaciones escalables y mantenibles. Fastapps no le tiene miedo a los ng-*. Para más información visita Angularjs website

Foundation

Less

Ya es imposible escribir CSS sin un preprocesador. Fastapps usa LESS para facilitar el cambio de colores de toda el app. Si necesitas mas información sobre less vente pa' ca' o aca tambien.

jQuery

Tienes algun plugin de jQuery que no puedes dejar de usar? Fastapps es compatible con jQuery. Solo ten cuidado de no usarlo erroneamente. De hecho el bundle ya tiene varios plugins integrados.

Gruntjs

Fastapps ya tiene integrado Gruntjs para que nos sea muy facil compilar los archivos LESS a CSS. Tambien podemos concatenar y minificar nuestos archivos javascript; esto nos permite tener una estructura de carpetas libres. PUM! algo cambió? Grunt compila todo. Ver más info En la docu

##Prerequisitos - Node / npm - Grunt CLI ##Instalación Para instalar o usar este boilerplate simplemente clona este repositorio dando `git clone https://github.com/juanpasolano/angular-mobile-demo.git` en la consola. Una vez clonados los archivos debes instalar las dependencias de node para que gruntjs funcione. Esto se hace dando `npm install` en la consola. Si quieres ver las dependencias que se estan usando, mira le archivo `package.json`. Luego de instalar los paquetes de node puedes comenzar a usar grunt dando `grunt` en la consola. ##Deploy a phonegap Para enviar los archivos a las carpetas de phonega es probable que tengas que modificar el archivo `Gruntfile.js` que esta en la raiz del directorio. En grunt esta disponible el task `copy-pg`. Este es el encargado de copiar los archivos hacia la carpeta de phonegap. Para que funcione bien tienes que cambiar la carpeta a la que esta apuntando `Gruntfile.js` ``` copy: { phonegap: { files: [ {expand: true, src: ['www/**'], dest: '../NOMBRE_CARPETA/platforms/ios/'}, {expand: true, src: ['www/**'], dest: '../NOMBRE_CARPETA/platforms/android/assets/'}, ] } } ``` ##Gruntjs Hay 3 tareas de grunt importantes en el boilerplate: - Comilar LESS. - Compilar y concatenar javascript. - Copiar archivos a carpetas de phonegap ##Less Estas son las variables de less disponibles ``` @mainFont:'Lato', Sans-serif; @sideBarWidth:250px; @headerHeight: 70px; @footerHeight:70px; @mainColor:#E6375A; @secondColor:spin(@mainColor, 20%); @darkGray:#4A4D4E; @lightGray:#dfe2e2; @headerColor:@darkGray; @sideNavColor:#fff; @footerColor:@darkGray; @contentColor: white; @paragraphColor:@darkGray; @headersColor:@darkGray; ```
